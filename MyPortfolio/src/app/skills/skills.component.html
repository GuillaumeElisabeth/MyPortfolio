C#/.NET

Microsoft .NET est le nom donné à l'ensemble des produits et de technologies informatiques de Microsoft pour rendre des applications facilement portables sur Internet. On utilise ces outils pour développer des logiciels, des sites web et des applications mobiles.

J'utilise les technologies Microsoft dans tous mes projets professionnels et personnels. Cela me permet de développer des applications en ayant une base déjà implémenter et donc plus facile à déployer pour tous types de plateformes.

Je connais bien les technologies Microsoft et certaines de leurs limites. Aujourd'ui je développe pour des clients d'Afrique des applications via Microsoft .NET car cela simplifie le déploiement sur place pour eux.

Pour évoluer dans ma carrière, je devrais m'intéresser plus aux technologies libres de droits ou autres que Microsoft.

Javascript

Javascript est un langage de programmation qui permet d'ajouter du comportement complexe à un site internet. Mais il permet aussi de communiquer avec des serveurs lorsqu'il est utilisé avec Node.js.

J'utilise Javascript dans tous les projets de site web car il me permet d'ajouter du comportement et de gérer le contenu de mon site dynamiquement. Ce langage de programmation est l'un des plus utilisés au monde.

Je connais bien Javascript et ses problématiques. Aujourd'hui, je développe une application web pour mes clients en Afrique car du fait de leur faible réseau informatique, on doit optimiser les échanges entre l'application et les serveurs distants. Javascript me permet de gérer localement beaucoup d'événements pour ne pas surcharger le réseau.

Pour évoluer dans ma carrière, je devrais m'intéresser aux nouvelles technologies du web comme Typescript et Angular (que j'utilise pour ce site internet), mais aussi d'autres outils comme React.

SQL Server Management Studio et Transact-SQL

SSMS est un système de gestion de base de données relationnel de Microsoft qui permet de gérer de la donnée. T-SQL est un langage de programmation associé aux bases de données relationnelles.

J'utilise SSMS dans mes projets professionnels car cette application offre une large gamme d'action et qu'elle est facile à prendre en main. De ce fait, lorsque je travaille avec des non techniques, il est très simple pour eux de reproduire une procédure via un tutoriel en ligne. T-SQL me permet d'écrire des scripts pour manipuler les données.

Je connais très bien SSMS et T-SQL car je les utilise tous les jours depuis 2 ans maintenant. Mon niveau technique sur ces technologies est très apprécié par mon équipe.

Pour évoluer dans ma carrière, je devrais m'intéresser aux autres types de base de données et passer une certification Microsoft SQL Serveur.

Unity

Unity est un moteur de jeu multi-plateforme. IL est l'un des plus répandus dans l'industrie du jeu vidéo. Cet outil m'a permis de développer des jeux complexes en nous proposant des articles préfabriqués que l'on peut utiliser gratuitement.

J'utilise Unity3d pour développer des jeux mais aussi pour travailler sur des aspects techniques très complexes comme le pathfinding, le multijoueur, la 3D, les animations... C'est un outil très puissant et très bien documenté.

Mon niveau sur Unity3D me permet de concevoir des petits jeux multijoueur avec des interactions, des objectifs et un peu d'animations. En 3ème année d'école, j'ai développé un jeu style action RPG sur Android dont le but était de parcourir un univers Steampunk et de résoudre des énigmes.

Pour évoluer sur cette technologie, je devrais travailler sur un projet de jeux avec une vision de déploiement sur l'application play Store.

HTML5 est un langage qui permet de concevoir des sites web alors que le CSS3 le mettra en forme (police, couleur, dimension,...). On utilise ces languages pour concevoir des sites internet.

L'utilité de HTML5 et CSS3 dans la création des sites internet est pour moi indéniable. Dès que je dois concevoir des sites internet, j'utilise ces outils. Ils sont à la base de tous les sites internet du monde.

Mon niveau de compétence est très élevé dans sur ces technologies. Je connais très bien ces outils et je les utilise tous les jours.

Pour évoluer sur ces technolgies, je devrais pousser encore plus loin dans la compréhension fondamentale de comment fonctionnent ces languages au plus bas niveau.

Talend data integration

Talend data intégration est un outil d'intégration de données servant à connecter, consulter et transformer tous les types de données depuis et vers n'importe quelle source.

J'utilise Talend dans mon projet à la société générale et elle me permet de collecter des données provenant de plusieurs sources à la fois, de les consulter, de les transformer et de les renvoyer vers d'autres sources. Cet outil me permet de faire travailler notre application avec des outils de la banque sans lien direct.

Mon niveau de compétence est très élevé dans sur cette technologie. Aujourd'hui je m'occupe directement d'une demande d'une filiale dont la majorité de la solution se base sur Talend.

Pour évoluer sur cette technolgie, je devrais pousser encore plus loin dans la compréhension fondamentale de comment fonctionne cet outil.
